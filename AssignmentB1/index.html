Project Name: WorldSurf

Project Description

Thank you for taking an interest in Web Interface and Usability. This site was created to store links to the direct information for usability tests and their respective results about WorldSurf web and desktop interfaces. The usability tests that will be conducted are in relation to navigational flow. Using the Usability Hub website, different mobile and desktop interfaces will be uploaded to get real-time responses from actual users. The test participants will be viewing various screens and given a task to perform with a goal in mind. Therefore, they will have to click the location on the screen in which they think is the correct process to advance through a pagenation process (i.e. filling out a form to sign up for a newsletter) and submitting the information.

New usability tests are posted every week. Please check back frequently to see what new results we have gathered!
